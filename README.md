 🔐 Port Scanner with Discord Alerts

This project is a CLI-based TCP port scanner built using Python, extended with real-time Discord webhook alerts.

 🚀 Features
- Scans a target IP over a range of ports
- Detects open ports using TCP connections
- Sends real-time alerts to Discord
- Provides a summary of open ports after scanning
 🧠 How it works
The tool attempts to connect to each port on the target system. If the connection is successful, the port is marked as OPEN and a message is sent to a Discord channel using webhooks.

 🧪 Example Output
Port 21: OPEN  
Port 22: OPEN  
Port 80: OPEN  

 ⚙️ Tech Stack
- Python  
- Socket Programming  
- CLI (Command Line Interface)  
- Discord Webhooks  

📌 Future Improvements
- Multithreading for faster scanning  
- Service/Banner detection  
- UDP scanning
